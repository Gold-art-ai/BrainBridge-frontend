# 🌉 BrainBridge

**BrainBridge** is a premium university project portfolio platform designed to showcase student innovation, facilitate cross-disciplinary connection, and bridge the gap between academia and industry.

> "Where developer projects come to life."

## 🚀 Features

### 📁 Project Portfolio
Showcase your work with rich project pages featuring deep technical stacks, field-specific tags, and media galleries. Demonstrate your expertise with precisely categorized project data.

### 🤝 Collaboration Hub
Find teammates by skills and interests. Tag collaborators, share repositories, and build together across 14 distinct innovation fields.

### 📈 Discovery Feed & Analytics
- **Discovery Feed**: A social-style feed that surfaces trending projects.
- **Analytics Dashboard**: Track views, engagement, and field rankings to understand how your projects perform.

### 🌍 Strategic Alignment
Unique integration for mapping projects to:
- **UN Sustainable Development Goals (SDGs)**: 17 goals covered.
- **Rwanda's NST2 Strategy**: Demonstrating real-world impact within local and global frameworks.

### 🛠️ Multi-Field Taxonomy
Precisely categorize projects across 14 fields including:
- Agriculture, Healthcare, AI, Cybersecurity, Finance, Smart Cities, and Sustainability.
- Supporting **150+ technology tags** and 13 main tech domains.

## 💻 Tech Stack

### Frontend
- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (Cinematic staged reveals and 3D parallax effects)
- **Icons**: [Lucide React](https://lucide.dev/)

### State & Real-time
- **State Management**: [Redux Toolkit](https://redux-toolkit.js.org/)
- **Real-time**: [WebSockets](https://developer.mozilla.org/en-US/docs/Web/API/WebSockets_API) (via `WebSocketProvider`)

### Tools & Versioning
- **Versioning**: Changesets
- **Linting**: ESLint

## 🛠️ Getting Started

### Prerequisites
- Node.js 18.18.0+
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/BrainBridge.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
